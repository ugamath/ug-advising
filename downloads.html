FAQ

Downloads
Why can’t I upload a file and have it appear publicly right away?

GitHub Pages is static hosting. It can serve files that are already in the repository, but it cannot store new uploads from site visitors without an additional backend service.

How do we publish a new template (PDF/Word) for students to download?

Add the file to the repository folder downloads/ and then add a link to it in the “Public downloads” list above.

What file types are allowed?

Common document/image formats are accepted. Keep the total upload size under 5 MB for the form. For larger files, email uga.math@snu.edu.in.
