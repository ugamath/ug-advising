What we handle here

  • Documentation (course catalogue, departmental documents, and documentation workflow)
  • Clarifications about course offerings and administrative course information

What we do not handle here

  • Exams/grades/timetable issues (use the Exams tab)
  • Minor/specialization decisions (use the Minor tab)
  • UG thesis policy/timeline matters (use the UG Thesis tab)
Tip: When submitting the form, include your SNU ID (if applicable), your programme/year, the course codes (if relevant), and any deadlines.

FAQ

Common questions
Where can I find the latest course catalogue?

If the catalogue is not visible on this website or the department portal, submit a query and mention the programme and semester.

A document needs correction (name, course list, etc.). What should I do?

Submit a documentation query and attach the document with a clear description of what needs to be corrected.

Do you handle internship approval letters?

If it is a departmental coordination step, submit here. If the process is entirely through a central university office, we will point you to the right workflow.
