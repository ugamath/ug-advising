UG Advising

Centralized platform for undergraduate advising and communication.

Last updated: 22 Dec 2025

How to use this website

  1. Pick the tab that matches your query.
  2. Read the scope + checklist.
  3. Prepare the required info before emailing the responsible faculty.

If your query doesn’t fit any tab, use “Other UG Academic/Advisory Matters”.

Minor, Specialization, and Post-Registration Course Concerns

Responsible faculty Dr. Charu

Scope

  • Queries related to Minor programs and Specializations
  • Concerns after course registration requiring faculty-level discussion
  • Excludes examination- and grade-related matters

Before you write

  • Student name + ID + program + batch
  • Minor/Specialization name and current status
  • Courses involved (course code + section)
  • Clear question + what outcome you want

Suggested subject: [UG Advising] Minor/Specialization Query — <StudentID>

Examination, Grades, Timetable, and Pre-Enrollment Queries

Responsible faculty Dr. Parul

Scope

  • Missed examinations and related concerns (often with Dean’s office)
  • Grade-related concerns and suggestions
  • Timetable management issues

Before you write

  • Course code + instructor + semester
  • Exact issue (missed exam / grade / timetable)
  • Any supporting document (medical note, email trail, etc.)
  • Deadline sensitivity (dates)

Suggested subject: [UG Advising] Exam/Grade/Timetable — <CourseCode> — <StudentID>

Academic Documentation and Course Administration

Responsible faculty Dr. Qazi Azhad

Scope

  • Documentation queries (course catalogue, etc.)
  • Course allotment and course offerings
  • Student internship, student activities

Before you write

  • What document/info you need (and why)
  • Program + semester + relevant course list
  • For internships/activities: dates + approvals needed

Suggested subject: [UG Advising] Documentation/Course Admin — <Topic> — <StudentID>

Undergraduate Thesis

Responsible faculty Dr. Neha

Scope

  • All matters related to UG projects and thesis work
  • Policies, timelines, supervision-related issues, student queries

Before you write

  • Project title / area + supervisor (if assigned)
  • Current stage (proposal / mid / final)
  • Timeline constraints + deliverables
  • Specific question (policy, evaluation, supervision, etc.)

Suggested subject: [UG Advising] Thesis/Project — <StudentID> — <Stage>

Any Other UG Academic or Advisory Matter

Responsible faculty Dr. Pradip

Scope

  • Any UG academic/advisory matter not explicitly covered in the other four categories

Before you write

  • Short description of the issue + why it doesn’t fit other tabs
  • Student info + program + semester
  • What you already tried / whom you contacted

Suggested subject: [UG Advising] Other Query — <StudentID> — <Topic>

This structure follows the UG Committee resolutions on responsibilities and the decision to create a centralized UG advising website.
